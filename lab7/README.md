# Лабораторна робота №7 — Перший API на Nest.js

## Опис
Task Manager API — серверний застосунок на Nest.js для управління задачами. 
Реалізовано згідно з принципами розподілу відповідальностей між Service та Controller.

## Технології
- **Nest.js** (TypeScript)
- **@nestjs/config** (для роботи з .env)
- **class-validator** & **class-transformer** (для валідації DTO)

## Ендпоінти

| Метод  | URL                      | Опис                          | Статус (успіх) |
| :----- | :----------------------- | :---------------------------- | :------------- |
| GET    | `/tasks`                 | Отримати всі задачі           | 200 OK         |
| GET    | `/tasks/search?status=..`| Фільтрація за статусом        | 200 OK         |
| GET    | `/tasks/:id`             | Отримати задачу за ID         | 200 / 404      |
| POST   | `/tasks`                 | Створити нову задачу          | 201 Created    |
| PATCH  | `/tasks/:id`             | Оновити поля задачі           | 200 / 404      |
| DELETE | `/tasks/:id`             | Видалити задачу               | 204 No Content |

## Запуск
1. Встановіть залежності: `npm install`
2. Налаштуйте оточення: скопіюйте `.env.example` у `.env` та вкажіть порт.
3. Запустіть сервер: `npm run start:dev`
